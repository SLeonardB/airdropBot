# Telegram Airdrop Bot Version 1
  Usage:
    This code creates a telegram bot to automate an airdrop giveaway at an Initial Coin Offering(ICO) to users with at least a verified wallet and a referral.
    Users receive more airdrops for referring more users if the referred users join before the deadline.

# Any Other Business
  The code can be altered to meet particular use cases.
  The sample ICO is for FlowMint(FM) Coin.
  ICO Channel link: https://t.me/flowmintcoin
  Telegram Bot username: @FlowMintDropBot
  This code is for demonstration purposes only.
  Email improvements to sleonardbilly23@gmail.com

# How to run the bot
# With python and pip installed, tested on Linux Termux
  python -m venv botEnv
  source botEnv/bin/activate
  pip install python-telegram-bot
  python fmbot_v1.py
