# Telegram Airdrop Bot Version 1
  _This code creates a telegram bot to automate an airdrop giveaway at an Initial Coin Offering(ICO) to users with at least a verified wallet and a referral.
  Users receive more airdrops for referring more users if the referred users join before the deadline._

  ## Running the bot, tested on _Linux, Termux_
  ### With python and pip installed 
  ```
  $ python -m venv botEnv
  $ source botEnv/bin/activate
  $ pip install python-telegram-bot
  $ python fmbot_v1.py
  ```
